🚀 DigiTools – Digital Tools Marketplace
📖 Overview
DigiTools is a modern, highly responsive web application that delivers a curated marketplace of premium digital tools. Designed with a sleek glassmorphic interface and vibrant visual aesthetics, the platform ensures an engaging and intuitive user experience.

Users can effortlessly explore dynamically loaded products, manage selections through an interactive cart system, and experience a smooth, simulated checkout flow — all within a fast and seamless interface.

⚙️ Technology Stack
Frontend: React.js (Vite)

Styling: Tailwind CSS v4, DaisyUI

Language: JavaScript (ES6+)

Notifications: React-Toastify

✨ Key Features
1. Dynamic Product Loading
Products are fetched from a structured products.json file, simulating real-world API integration and ensuring scalability for future backend connectivity.

2. Advanced Cart Management
Add products to cart with real-time updates

Prevent duplicate entries with intelligent validation

Instant removal of items with UI synchronization

Automatic total price calculation

3. Real-Time User Feedback
Integrated React-Toastify provides instant, visually appealing notifications for user actions such as adding items, duplicate warnings, and removals.

4. Seamless View Switching
Effortlessly toggle between:

Product Grid View (3-column responsive layout)

Cart View (clean and minimal interface)

All transitions occur without page reloads, ensuring a smooth single-page application experience.
